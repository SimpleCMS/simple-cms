The Static CMS exposes a `window.CMS` global object that you can use to customize your CMS. Available options are:

- Register [custom widgets](/docs/custom-widgets)
- Register [custom previews](/docs/custom-previews)
- Register [editor customizations](/docs/widget-markdown#customization)
- Register [external menu links](/docs/external-links)
- Register [custom pages](/docs/custom-pages)
